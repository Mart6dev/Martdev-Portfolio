# Martin Nsiah Portfolio V1

A stunning, modern portfolio website for Martin Nsiah (Mart6dev) - Web3 Architect & Software Engineer.

## 🚀 Features

- **Modern Glass-morphism Design** with animated backgrounds
- **Responsive Layout** that works perfectly on all devices
- **Interactive Elements** including magnetic cursor and smooth animations
- **Multiple Contact Methods** - WhatsApp, Email, Phone, LinkedIn
- **Professional Sections** - About, Skills, Projects, Experience, Education
- **SEO Optimized** with proper meta tags and descriptions
- **Fast Loading** with optimized assets and minimal dependencies

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Host on a web server** (Apache, Nginx, or static hosting)
3. **Access via browser** - no build process required

### Local Development

```bash
# Start a local server
python -m http.server 8000
# or
npx serve .
# or
php -S localhost:8000
```

Then visit `http://localhost:8000`

## 🔧 Customization

### Colors
Edit CSS variables in `:root`:
```css
:root {
    --boyos-orange: #E66C2C;
    --craft-purple: #7828C8;
    --success-green: #10B981;
    /* ... */
}
```
